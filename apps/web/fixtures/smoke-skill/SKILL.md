# Web Smoke Reviewer

Use this skill to review SkillHub UI changes. Focus on:

- Whether a user can create a Skill from a standard bundle.
- Whether runtime environment tags are captured on each eval run.
- Whether eval set case versions are visible and traceable.
- Whether manual pass/fail evaluation can be recorded without hidden steps.
